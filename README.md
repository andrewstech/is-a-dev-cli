# IS A DEV CLI

A new way to generate your free subdomains without messing around with json files.



## Authors

- [@andrewstech](https://www.github.com/andrewstech)


## Installation

Install the CLI with npm

```bash
  npm install @is-a-dev/cli -g
```

Run the CLI 

```bash
  is-a-dev
```

For commands type
```bash
  is-a-dev help
```
