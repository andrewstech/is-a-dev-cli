# is-a-dev CLI
A new way to generate your free subdomains from your teminal.

## Authors
- [@andrewstech](https://www.github.com/andrewstech)

## Installation
Install the CI with npm.

```bash
npm install @is-a-dev/cli -g
```

Run the CI

```bash
is-a-dev
```

To see all available commands, run:

```bash
is-a-dev help
```
